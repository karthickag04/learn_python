# Python Practice Task Set

This folder contains extra practice tasks based on the course folders in this repository.

Work through the files in order:

1. `01_foundation_tasks.py` - print, comments, variables, data types
2. `02_operator_control_tasks.py` - operators, if/elif/else, match-case style thinking
3. `03_data_structure_tasks.py` - lists, tuples, dictionaries, sets, strings
4. `04_loop_tasks.py` - for loops, while loops, nested loops, loop control
5. `05_function_tasks.py` - functions, return values, lambda, scope, recursion
6. `06_math_utility_tasks.py` - math module and utility calculations
7. `07_oop_tasks.py` - classes, constructors, methods, encapsulation
8. `08_advanced_oop_tasks.py` - inheritance, polymorphism, abstract class ideas
9. `09_exception_regex_file_tasks.py` - exceptions, regex, text files, CSV/JSON practice
10. `10_mini_project_tasks.py` - small project challenges based on the practice projects

How to use:

- Read the task comments.
- Replace each `pass` or placeholder value with your solution.
- Run a file with:

```bash
python task/01_foundation_tasks.py
```

Most files include a few simple `assert` checks inside `if __name__ == "__main__":`.
If the file runs without an error, your solution is likely correct for the given examples.
