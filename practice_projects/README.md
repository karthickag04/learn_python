This folder contains complete practice projects to reinforce your Python learning.

## Project 1: Student Management System
A complete CRUD application for managing student records.

## Project 2: Calculator Application

## Project 4: Quiz Game
Interactive quiz with scoring and categories.

Each project file is self-contained and can be run independently.
Read the comments in each file for instructions and explanations.
